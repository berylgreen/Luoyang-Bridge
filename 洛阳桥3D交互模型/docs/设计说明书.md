# 设计说明书

## 《洛阳桥3D交互模型》— 泉州洛阳桥三维结构展示

---

## 一、项目概述

基于WebGL的交互式三维模型展示系统，以参数化建模方式还原泉州洛阳桥的五层工程结构，支持360°旋转、结构分层、视角切换和知识点讲解。

## 二、3D建模方案

### 2.1 参数化建模
全部几何体由Three.js程序化生成：
- **船形桥墩**：使用 `ExtrudeGeometry` 从二维船形截面拉伸
- **筏型基础**：`BoxGeometry` + 随机散布石块
- **种蛎固基**：`BoxGeometry` + `SphereGeometry` 牡蛎簇
- **石板梁**：5×5 `BoxGeometry` 阵列
- **栏杆石狮**：柱体 + 球体组合

### 2.2 程序化纹理
使用Canvas API生成四种纹理贴图（石材、深色石材、牡蛎层、水面法线），无需外部图片文件。

## 三、交互设计

- **结构分层**：点击底部按钮高亮特定部件，其余透明化
- **视角切换**：正面/侧面/俯视/剖面/透视五种预设视角
- **双击选中**：射线检测(Raycaster)实现3D物体点选
- **知识面板**：右侧面板实时展示当前结构的工程知识
