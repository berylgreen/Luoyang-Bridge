# 洛阳古渡·万安长虹 — 泉州洛阳桥数字文化展板

> 2026年（第19届）中国大学生计算机设计大赛（4C）参赛作品
> 第七大类：数媒静态设计

## 📖 项目简介

以中国传统长卷画的视觉语言，采用**横向滚动叙事**方式，展现泉州洛阳桥的千年建造传奇。整体视觉取法宋代绢帛画卷，色调以绢帛米黄为底、墨色为文、朱砂为印，兼具学术严谨与传统美学。

## 📜 六面展板

| 面板 | 内容 | 设计特色 |
|------|------|---------|
| 1 卷首 | 标题 + 年代 + 印章 | 传统竖排题字 + 朱印 |
| 2 造桥者 | 蔡襄生平 + 引言 | 肖像框 + 工笔排版 |
| 3 四大工法 | 筏基/种蛎/船墩/浮运 | 四宫格图解卡片 |
| 4 结构剖面 | SVG六层桥梁截面图 | 工程制图 + 数据标注 |
| 5 文化传承 | 文保/世遗/诗词 | 成就卡片 + 题诗框 |
| 6 卷尾 | 参考文献 + 印鉴 | 学术引用 + 圆印 |

## 🚀 启动方式

```bash
# 直接双击 index.html 即可
# 或使用本地服务器
python3 -m http.server 8080
```

操作：**鼠标滚轮**横向浏览 | **左右方向键** | 底部**导航点**跳转
