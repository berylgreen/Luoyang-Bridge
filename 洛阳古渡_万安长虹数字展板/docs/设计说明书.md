# 设计说明书

## 《洛阳古渡·万安长虹》— 泉州洛阳桥数字文化展板
### 2026年（第19届）中国大学生计算机设计大赛 · 第七大类 · 数媒静态设计

---

## 一、选题依据

泉州洛阳桥（万安桥），中国第一座跨海大石桥，由北宋蔡襄主持建造（1053-1059年），2021年入选UNESCO世界遗产。其"筏型基础"与"种蛎固基"技术比西方同类工法早约700年。

## 二、设计理念

### 2.1 核心概念
**"以长卷之形，载千年之桥"** — 借鉴中国传统横卷画的叙事形式，以**横向滚动**的数字长卷承载洛阳桥的文化内涵。

### 2.2 视觉风格
取法**宋代绢帛画**的审美意蕴：
- **底色**：绢帛米黄（#f0e6d0）— 模拟传世古卷的绢帛质感
- **正文**：墨色（#2a2018）— 体现书法运笔的浓淡韵律
- **强调**：朱砂红（#c23a2a）— 传统印章与圈点的经典用色
- **辅色**：靛蓝（#2a4a6a）— 呼应海洋工程的深沉底蕴

### 2.3 字体选择
- **标题**：Ma Shan Zheng（马善政楷书）— 呼应宋代碑帖的庄重典雅
- **正文**：Noto Serif SC（思源宋体）— 保证长文阅读的清晰舒适

## 三、展板内容

六面展板采用"起承转合"的叙事结构：
1. **卷首**：传统竖排题字 + 红色印章，营造开卷即古韵的第一印象
2. **造桥者**：蔡襄生平与建桥故事
3. **四大工法**：核心创新技术的图文解说
4. **结构剖面**：SVG矢量绘制的六层桥梁截面工程图
5. **文化传承**：文保与世遗成就
6. **卷尾**：参考文献 + 收卷印鉴

## 四、交互设计

- **横向滚动**：鼠标滚轮驱动，模拟展开长卷的物理体验
- **视差动画**：远景山峦与云雾以不同速率移动，增强空间深度
- **渐显效果**：各面板元素在进入视口时依序显现
- **导航系统**：底部圆点导航 + 顶部进度条
