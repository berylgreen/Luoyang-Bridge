# 跨海飞虹 — 泉州洛阳桥古代海洋工程交互图鉴

> 2026年（第19届）中国大学生计算机设计大赛（4C）参赛作品
> 第六大类：信息可视化设计

## 📖 项目简介

以交互式数据可视化网页的形式，深度解析中国第一座跨海大石桥——泉州洛阳桥的四大工程创举：
- 🪨 **筏型基础** — 比西方同类技术早约700年
- 🦪 **种蛎固基法** — 人类首个生物+工程融合的大型基建
- ⛵ **船形桥墩** — 古代流体力学的精妙实践
- 🌊 **浮运架梁法** — 以天地之力代替人工起重

## 🚀 快速启动

### 方式一：直接打开
双击 `index.html` 即可在浏览器中查看（推荐使用 Chrome/Firefox/Edge）。

### 方式二：本地服务器
```bash
# 使用 Node.js
npx serve .

# 或使用 Python
python3 -m http.server 8080
```

## 📁 项目结构

```
跨海飞虹_洛阳桥交互图鉴/
├── index.html              # 主页面（7大篇章完整交互体验）
├── css/
│   ├── base.css            # 设计系统（配色/字体/间距/变量）
│   ├── layout.css          # 布局系统（栅格/导航/响应式）
│   ├── components.css      # UI组件（卡片/时间线/图表容器等）
│   └── animations.css      # 动画系统（滚动触发/波浪/粒子）
├── js/
│   ├── data.js             # 数据模块（严谨考据的历史数据）
│   ├── charts.js           # ECharts图表配置（6个可视化图表）
│   ├── scroll-animations.js# 滚动动画系统
│   ├── bridge-interactive.js# 桥梁结构SVG交互
│   └── main.js             # 主控逻辑
├── docs/
│   ├── 设计说明书.md        # 参赛设计说明文档
│   └── AI工具使用报告.md    # AI工具合规声明
└── README.md               # 本文件
```

## 🎨 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 + JavaScript | 核心开发 |
| ECharts 5.5 | 数据可视化图表 |
| Noto Serif/Sans SC | 中文字体 |
| Intersection Observer API | 滚动驱动动画 |

## 📊 包含的可视化图表

1. 宋代泉州港贸易数据（柱状图+折线图）
2. 工程综合评估雷达图（洛阳桥 vs 同期西方）
3. 桥梁结构材料占比（环形图）
4. 洛阳江口潮汐模拟（折线图）
5. 古今桥梁长度对比（条形图）
6. 千年修缮历程（散点图）
