# 设计说明书

## 《古法造桥：洛阳桥建造模拟器》

### 2026年（第19届）中国大学生计算机设计大赛
### 第九大类：数媒游戏与交互设计

---

## 一、选题依据

### 1.1 选题背景
泉州洛阳桥是中国第一座跨海大石桥，由北宋蔡襄主持建造（1053-1059年）。其四大工程创举——筏型基础、种蛎固基、船形桥墩、浮运架梁——在世界桥梁工程史上具有里程碑意义。2021年入选UNESCO世界文化遗产。

### 1.2 选题理由
本作品以交互式教育游戏的形式，让玩家亲身体验建造洛阳桥的全过程，在游戏中学习古代建筑工程知识。选择第9大类（游戏与交互设计）赛道，旨在通过"寓教于乐"的方式传播中国古代建筑文化。

---

## 二、游戏设计

### 2.1 核心理念
**"在建造中理解，在游戏中传承"** — 每个关卡对应洛阳桥建造的一个关键工序，玩家通过完成游戏操作来亲身体验古代工匠的智慧。

### 2.2 关卡设计

| 关卡 | 工程阶段 | 游戏机制 | 教育目标 |
|------|---------|---------|---------|
| 1 | 观潮选址 | 节奏判断（在低潮位点击） | 理解半日潮规律 |
| 2 | 抛石筑基 | 拖拽放置（石块到目标区） | 理解筏型基础 |
| 3 | 种蛎固基 | 点击培育（牡蛎生长系统） | 理解生物工程 |
| 4 | 砌筑桥墩 | 拼图组装（船形结构） | 理解流体力学 |
| 5 | 浮运架梁 | 时机把控（潮汐升降） | 理解浮运架梁法 |

### 2.3 评分系统
每关根据操作精确度给予积分，五关总分汇总。通关后展示"知识回顾"界面，系统性讲解五大工程原理。

---

## 三、技术实现

### 3.1 技术架构

- **渲染引擎**：HTML5 Canvas 2D API
- **游戏循环**：requestAnimationFrame 驱动的 60fps 主循环
- **输入系统**：鼠标/触摸统一处理（支持移动端）
- **状态机**：menu → levelIntro → playing → dialog → complete
- **无框架依赖**：纯原生 JavaScript 实现

### 3.2 性能优化
- Canvas 逐帧清除重绘
- 粒子系统使用对象池
- 触摸事件使用 passive 优化

---

## 四、视觉设计

- **配色方案**：延续宋韵+海洋工程的深海靛蓝主调
- **UI风格**：玻璃态（Glassmorphism）卡片 + 金色强调色
- **字体**：Noto Serif SC（标题）+ Noto Sans SC（正文）
- **动画**：关卡过渡淡入、分数弹出、水波纹效果
