# 古法造桥 — 洛阳桥建造模拟器

> 2026年（第19届）中国大学生计算机设计大赛（4C）参赛作品
> 第九大类：数媒游戏与交互设计

## 📖 项目简介

一款教育性质的交互式网页游戏，让玩家扮演北宋工匠，在蔡襄（泉州知府）的指挥下，亲手体验建造中国第一座跨海大石桥——泉州洛阳桥的五大工程创举。

## 🎮 五个关卡

| 关卡 | 名称 | 玩法 | 科普的工程原理 |
|------|------|------|--------------|
| 1 | 🌊 观潮选址 | 观察潮汐节律，在最低潮位时点击 | 半日潮规律 |
| 2 | 🪨 抛石筑基 | 拖拽石块到水中目标区域 | 筏型基础 |
| 3 | 🦪 种蛎固基 | 点击石缝播撒牡蛎种苗 | 生物胶结工程 |
| 4 | ⛵ 砌筑桥墩 | 拼装船形桥墩结构 | 流体力学分水 |
| 5 | 🏗️ 浮运架梁 | 借潮汐之力精准放置石板梁 | 浮运架梁法 |
